A Flask API for cricket score calculation with web scraping capabilities

Base URL: https://your-app.onrender.com

API Endpoints

GET /api/health

Check API health status.

Response:
{"status": "healthy", "service": "Cricket Score API", "version": "1.0"}
GET /api/matches

Get list of current cricket matches (scraped from external sources).

Example Request:
GET /api/matches

Example Response:
{"status": "success", "count": 3, "matches": [...]}
GET /api/match/<match_id>

Get detailed information about a specific match.

Example:
GET /api/match/match_001
POST /api/calculate/player

Calculate fantasy points for a player.

Request Body (batsman example):
{ "player_type": "batsman", "runs": 65, "balls_faced": 42, "fours": 8, "sixes": 2, "dismissal_type": "caught" }

Response:
{"status": "success", "player_type": "batsman", "points": 98.5}

Deployment to Render

  1. Create a new account on Render.com
  2. Click "New +" and select "Web Service"
  3. Connect your GitHub/GitLab repository or deploy manually
  4. Set the following:
    • Name: cricket-score-api
    • Runtime: Python 3
    • Build Command: pip install -r requirements.txt
    • Start Command: gunicorn app:app
  5. Click "Create Web Service"
  6. Your API will be available at: https://cricket-score-api.onrender.com
