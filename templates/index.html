Использование API

Наше API предоставляет доступ к актуальной статистике крикета через RESTful интерфейс:

Пример запроса:

GET /api/v1/matches/live
GET /api/v1/players/top?by=runs&limit=10
GET /api/v1/teams/1/stats

Пример ответа:

{
  "matches": [
    {
      "id": 1,
      "team1": "India",
      "team2": "Australia",
      "status": "completed",
      "result": "India won by 37 runs"
    }
  ]
}
