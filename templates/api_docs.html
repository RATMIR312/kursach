Документация API

Полное описание REST API для доступа к данным о крикете

Основная информация

  • Базовый URL: /api/v1
  • Формат данных: JSON
  • Кодировка: UTF-8

Доступные эндпоинты

GET /api/v1/health

Проверка работоспособности API

GET /api/v1/matches

Получение списка матчей

Параметры:
  • type - тип матча (Test, ODI, T20)
  • status - статус (live, completed, scheduled)
GET /api/v1/matches/<id>

Получение детальной информации о матче

GET /api/v1/teams

Получение списка команд

GET /api/v1/teams/<id>

Получение статистики команды

GET /api/v1/players

Получение списка игроков

GET /api/v1/players/<id>

Получение статистики игрока

GET /api/v1/players/top

Получение лучших игроков

Параметры:
  • by - критерий (runs, wickets)
  • limit - количество (по умолчанию: 10)
POST /api/v1/admin/scrape

Запуск скрапинга данных (требуется аутентификация)

Примеры использования

Пример 1: Получение всех матчей

fetch('/api/v1/matches')
  .then(response => response.json())
  .then(data => console.log(data));

Пример 2: Получение статистики команды

fetch('/api/v1/teams/1')
  .then(response => response.json())
  .then(data => console.log(data));

Пример 3: Получение лучших бэтсменов

fetch('/api/v1/players/top?by=runs&limit=5')
  .then(response => response.json())
  .then(data => console.log(data));
